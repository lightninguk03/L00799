# Project Neon 前端开发文档 (更新版 V2.6)

## 1. 项目概述
本项目是一个基于赛博朋克风格的 ACG 社交元宇宙平台前端，名为 "LETAVERSE (莱塔宇宙)"。项目结合了最新的后端 V2.5.1 能力，提供完整的社交互动体验，支持后台动态配置网站内容。

### 核心特性 (V2.6 Updated)
- **极致赛博朋克视觉**: 
  - 全局深色主题，霓虹紫 (#8a2be2) 和赛博青 (#00ffff) 配色
  - 毛玻璃效果、发光边框、动态渐变
  - AI 助手 "Mu AI" 悬浮聊天窗口
- **完整的社交互动**:
  - **动态管理**: 发布、编辑、删除、转发动态
  - **通知系统**: 点赞、评论、关注、转发通知，支持全部已读
  - **全站搜索**: 搜索动态和用户
  - **关注系统**: 关注/取消关注用户，查看关注列表和粉丝列表
- **用户个性化**:
  - 头像上传 (支持图片预览)
  - 用户名修改
  - 个人数据统计
- **认证系统**:
  - 双 Token 认证 (access_token + refresh_token)
  - 邮箱验证流程
  - 忘记密码/重置密码
  - 路由保护 (RequireAuth)
  - 封禁检测与提示页面
- **管理功能** (V2.4 新增):
  - 系统公告展示 (首页顶部轮播)
  - 举报功能 (动态/评论/用户)
  - 封禁状态检测与提示

## 2. 技术栈
- **核心框架**: React 18 + TypeScript
- **构建工具**: Vite
- **样式方案**: Tailwind CSS
- **状态管理**: TanStack Query (React Query)
- **路由**: React Router DOM v6
- **国际化**: i18next (中/英双语)
- **动画**: Framer Motion
- **HTTP 客户端**: Axios
- **图标**: Lucide React
- **通知**: React Hot Toast

## 4. 核心功能模块详解

### 4.1 认证系统
- **双 Token 机制**: access_token (30分钟) + refresh_token (7天)
- **自动刷新**: 401 错误时自动使用 refresh_token 刷新
- **路由保护**: `RequireAuth` 组件包裹需要登录的路由
- **登录状态**: `useAuth` Hook 提供统一的认证状态管理

```tsx
// 使用示例
const { isLoggedIn, user, requireAuth } = useAuth();

// 需要登录才能执行的操作
if (!requireAuth(undefined, '请先登录')) return;
```

### 4.1.1 站点配置 (SiteConfigContext) - V2.5 重大更新
- **自动刷新**: 每 5 分钟自动从 `/system/config` 获取最新配置
- **页面可见性**: 用户切换回页面时自动刷新
- **手动刷新**: 提供 `refreshConfig()` 方法
- **动态配置**: 支持后台管理修改网站品牌、AI 信息、视觉资源等

```tsx
// 使用示例
const { config, refreshConfig } = useSiteConfig();
const { i18n } = useTranslation();
const isZh = i18n.language.startsWith('zh');

// 获取品牌信息 (支持中英双语)
const siteName = isZh ? config.siteNameCn : config.siteName;
const slogan = isZh ? config.sloganCn : config.slogan;

// 获取 AI 信息
const aiName = isZh ? config.aiNameCn : config.aiName;
const aiGreeting = isZh ? config.aiGreetingCn : config.aiGreeting;

// 获取视觉资源 (需要用 getMediaUrl 处理)
import { getMediaUrl } from '../lib/utils';
const logo = getMediaUrl(config.logo);
const background = getMediaUrl(config.backgroundImage);
const aiKanban = getMediaUrl(config.kanbanGirl);

// 手动刷新配置
await refreshConfig();
```

**可配置项列表**:
| 配置项 | 字段名 | 说明 |
|--------|--------|------|
| 网站名称 | siteName / siteNameCn | 英文/中文名称 |
| 社区名称 | communityName / communityNameCn | 英文/中文名称 |
| 品牌标语 | slogan / sloganCn | 英文/中文标语 |
| AI 名称 | aiName / aiNameCn | AI 助手名称 |
| AI 头衔 | aiTitle / aiTitleCn | AI 助手头衔 |
| AI 欢迎语 | aiGreeting / aiGreetingCn | AI 聊天欢迎语 |
| Logo | logo | 网站 Logo 图片 |
| 背景图 | backgroundImage | 全局背景图 |
| Hero 背景 | heroBackground | 首页 Hero 区背景 |
| AI 看板娘 | kanbanGirl | AI 立绘图片 |
| 社交链接 | socialLinks | 社交媒体链接数组 |

## 5. API 对接说明 (V2.4)

### 5.1 分页格式
所有列表接口返回统一格式：
```json
{
  "total": 100,
  "page": 1,
  "page_size": 20,
  "items": [...]
}
```

### 5.2 媒体 URL 处理
**V2.3 重要变更**: `media_urls` 现在直接返回数组，不再是 JSON 字符串。

```typescript
// 使用工具函数处理 (兼容新旧格式)
import { getMediaUrl, getAllMediaUrls, getAvatarUrl } from '../lib/utils';

// 获取第一张图片
const imageUrl = getMediaUrl(post.media_urls);

// 获取所有图片
const allImages = getAllMediaUrls(post.media_urls);

// 获取头像 URL
const avatarUrl = getAvatarUrl(user.avatar);
```

### 5.3 错误处理
```typescript
// 统一错误格式
{
  "error_code": "invalid_credentials",
  "message": "用户名或密码错误"
}

// 429 限流错误
{
  "error": "Rate limit exceeded: 5 per 1 minute"
}
```

### 5.4 限流保护 (V2.3 新增)
| 接口 | 限制 |
|------|------|
| POST /auth/login | 5次/分钟 |
| POST /auth/register | 3次/分钟 |
| POST /ai/chat | 10次/分钟 |
| POST /posts/ | 20次/分钟 |

### 5.5 管理后台 API (V2.4 新增)

```typescript
// 系统公告（公开）
GET /system/announcements
// 返回: Announcement[]

// 举报（需登录）
POST /reports
// Body: { target_type: 'post'|'comment'|'user', target_id: number, reason: string }

// 封禁检查（登录时自动）
// 返回 403 + { ban_reason: string, banned_until: string }
```

```json
GET /system/config

{
  "site_name": "LETAVERSE",
  "site_name_cn": "莱塔宇宙",
  "community_name": "Lightning Community",
  "community_name_cn": "闪电社区",
  "slogan": "The soul is infinite...",
  "slogan_cn": "灵魂无限...",
  "logo": "/uploads/images/logo.png",
  "background": "/uploads/images/bg.jpg",
  "hero_background": "/uploads/images/hero.jpg",
  "ai_kanban": "/uploads/images/kanban.png",
  "ai": {
    "name": "Mu AI",
    "name_cn": "穆爱",
    "title": "Central Brain",
    "title_cn": "中枢脑",
    "greeting": "Hello~ I'm Mu ✨",
    "greeting_cn": "你好呀～我是穆爱 ✨"
  },
  "social_links": [
    {"name": "Instagram", "url": "https://...", "icon": "instagram"}
  ]
}
```

### 前端配置映射
| 后端字段 | 前端字段 | 使用位置 |
|----------|----------|----------|
| site_name | siteName | 首页 Hero 标题 |
| site_name_cn | siteNameCn | 首页 Hero 副标题 |
| slogan | slogan | 首页标语 (英) |
| slogan_cn | sloganCn | 首页标语 (中) |
| logo | logo | NavBar Logo |
| background | backgroundImage | Layout 全局背景 |
| hero_background | heroBackground | 首页 Hero 背景 |
| ai_kanban | kanbanGirl | ChatWidget、首页 AI 立绘 |
| ai.name | aiName | ChatWidget、MuAI 页面 |
| ai.name_cn | aiNameCn | ChatWidget、MuAI 页面 |
| ai.greeting | aiGreeting | ChatWidget 欢迎语 |
| ai.greeting_cn | aiGreetingCn | ChatWidget 欢迎语 |

### 注意事项
1. **图片 URL 处理**: 后端返回的图片路径需要用 `getMediaUrl()` 处理，会自动添加后端基础 URL
2. **中英双语**: 根据 `i18n.language` 判断使用 `_cn` 后缀的字段还是英文字段
3. **默认值**: 如果后端配置为空，会使用 `defaultSiteConfig` 中的默认值
4. **自动刷新**: 配置每 5 分钟自动刷新，页面切换回来时也会刷新

## 12. 赛博朋克动态效果使用指南 (V2.6)

### CSS 动画类名速查表

| 类名 | 效果 | 适用场景 |
|------|------|----------|
| `neon-flicker` | 霓虹闪烁 | 标题、状态文字 |
| `neon-pulse` | 青色呼吸发光 | 按钮、图标容器 |
| `neon-pulse-purple` | 紫色呼吸发光 | 按钮、装饰元素 |
| `glitch-hover` | 悬停故障抖动 | 按钮、可点击元素 |
| `data-flow` | 数据流动画 | 背景、分割线 |
| `border-glow` | 边框呼吸 | 卡片、输入框 |
| `hologram` | 全息抖动 | AI 元素、头像 |
| `card-scan` | 卡片扫描线 | 卡片组件 |
| `light-trace` | 光线追踪 | 输入框、卡片 |
| `particle` | 浮动粒子 | 背景装饰 |
| `rgb-split-hover` | RGB分离悬停 | 标题、按钮 |
| `circuit-bg` | 电路板纹理 | 背景 |
| `cute-bounce` | 可爱弹跳 | 加载动画 |
| `heartbeat` | 心跳效果 | 状态指示、图标 |
| `gradient-border` | 渐变边框 | 分割线、边框 |
| `float-card` | 悬浮效果 | 卡片悬停 |
| `energy-core-btn` | 能量核心 | 主要按钮 |

### 使用示例

```tsx
// 霓虹闪烁标题
<h1 className="text-cyber-cyan neon-flicker">SYSTEM ONLINE</h1>

// 呼吸发光按钮
<button className="neon-pulse">点击</button>

// 悬停故障效果
<button className="glitch-hover">提交</button>

// 心跳状态指示
<span className="w-2 h-2 bg-cyber-cyan rounded-full heartbeat" />

// 卡片扫描线 + 悬浮效果
<div className="card-scan float-card">卡片内容</div>

// 背景粒子
<div className="particle" style={{ left: '10%', top: '20%', animationDelay: '0s' }} />

// RGB分离悬停
<span className="rgb-split-hover">文字</span>

// 渐变边框分割线
<div className="h-[2px] gradient-border" />

// 加载动画
<div className="flex space-x-1">
  <div className="w-2 h-2 bg-cyber-cyan rounded-full cute-bounce" style={{ animationDelay: '0s' }} />
  <div className="w-2 h-2 bg-neon-purple rounded-full cute-bounce" style={{ animationDelay: '0.1s' }} />
  <div className="w-2 h-2 bg-cyber-cyan rounded-full cute-bounce" style={{ animationDelay: '0.2s' }} />
</div>
```
