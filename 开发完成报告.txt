🚀 快速启动

Windows 用户:
  1. 双击运行 start.bat
  2. 编辑 .env 文件（首次运行会提示）
  3. 再次运行 start.bat

Linux/Mac 用户:
  1. chmod +x start.sh
  2. ./start.sh
  3. 编辑 .env 文件
  4. 再次运行 ./start.sh

访问 API 文档:
  http://localhost:8000/docs

🔧 技术栈

后端框架: FastAPI 0.104.1
ORM: SQLModel 0.0.14
数据库: SQLite (开发) / PostgreSQL (生产)
认证: JWT (python-jose)
密码加密: bcrypt (passlib)
AI 集成: OpenAI API 1.3.5
文件处理: aiofiles 23.2.1
Python 版本: 3.10+

📝 API 端点总览

认证 (/auth):
  POST /auth/register - 用户注册
  POST /auth/login - 用户登录

动态 (/posts):
  POST /posts/ - 发布动态
  GET /posts/ - 获取列表
  GET /posts/{id} - 获取详情
  POST /posts/{id}/interact - 点赞/收藏
  POST /posts/{id}/comments - 发表评论
  GET /posts/{id}/comments - 获取评论
  POST /posts/{id}/repost - 转发动态

图片 (/gallery):
  GET /gallery/categories - 获取分类
  GET /gallery/{category_id} - 获取分类图片

AI (/ai):
  POST /ai/chat - AI 对话

系统 (/system):
  GET /system/config - 系统配置
